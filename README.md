# DAEMON Landing Page

The marketing site and documentation for [DAEMON](https://github.com/nullxnothing/daemon), an open-source IDE built for AI-native Solana development.

**Live:** [daemon-landing.vercel.app](https://daemon-landing.vercel.app)

## Stack

- Next.js 15 (App Router)
- Tailwind CSS v4
- Deployed on Vercel

## Development

```bash
pnpm install
pnpm dev
```
