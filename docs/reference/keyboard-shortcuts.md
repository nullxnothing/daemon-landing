# Keyboard Shortcuts

## Essential Shortcuts

| Shortcut | Action |
| --- | --- |
| `Ctrl+K` | Command Drawer (access all tools) |
| `Ctrl+P` | Quick Open (file search) |
| `Ctrl+Shift+P` | Command Palette |
| `Ctrl+Shift+A` | Agent Launcher |
| `Ctrl+Shift+G` | Grind Mode |
| `Ctrl+,` | Settings |

## Editor

| Shortcut | Action |
| --- | --- |
| `Ctrl+S` | Save file |
| `Ctrl+Z` | Undo |
| `Ctrl+Shift+Z` | Redo |
| `Ctrl+F` | Find in file |
| `Ctrl+H` | Find and replace |
| `Ctrl+D` | Select next occurrence |
| `Ctrl+Shift+L` | Select all occurrences |
| `Ctrl+/` | Toggle line comment |
| `Alt+Up/Down` | Move line up/down |
| `Ctrl+Shift+K` | Delete line |

## Navigation

| Shortcut | Action |
| --- | --- |
| `Ctrl+P` | Quick Open (file search) |
| `Ctrl+Shift+B` | Browser Tab |
| `Ctrl+Shift+D` | Dashboard Tab |
| `Ctrl+Tab` | Next editor tab |
| `Ctrl+Shift+Tab` | Previous editor tab |

## Terminal

| Shortcut | Action |
| --- | --- |
| `` Ctrl+` `` | Toggle Terminal |
| `Ctrl+Shift+F` | Terminal Search |
| `Ctrl+R` | Reverse search history |

## AI Agents

| Shortcut | Action |
| --- | --- |
| `Ctrl+Shift+A` | Agent Launcher |
| `Ctrl+Shift+G` | Grind Mode |

{% hint style="info" %}
All shortcuts use `Ctrl` on Windows and `Cmd` on macOS.
{% endhint %}
