# Roadmap

DAEMON is actively developed. Here's what's coming next.

## In Development

### Multi-Agent Orchestration

Run multiple AI agents simultaneously with intelligent task distribution and coordination. Building on Grind Mode's foundation with smarter inter-agent communication.

### Local LLM Support

Run local language models for offline AI assistance with Ollama and LM Studio integration. Use open-source models when you don't need Claude's full capabilities, or work completely offline.

## Planned

### Real-Time Collaboration

Work with teammates in real-time with live cursors, presence indicators, and shared sessions. Bring the multiplayer experience to DAEMON.

### Enhanced Security

Advanced sandboxing, code signing verification, and secure credential management. Defense-in-depth for your development environment.

### Cloud Sync

Sync your settings, plugins, and workspace configurations across devices. Pick up where you left off on any machine.

### Team Workspaces

Shared project configurations, MCP servers, and agent templates for teams. Standardize your team's development environment.
